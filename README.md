# Tasks Manager

### 🛠️ Running the project

`git clone https://github.com/Itau-Dev-Boost-Grupo-1/tasks-manager`

`cd tasks-manager`

`yarn install`

### 🧑‍🎨 Running SASS

`yarn sass` to watch for .scss files and compile the output at **css** folder.
